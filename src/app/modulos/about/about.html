Acerca del Proyecto

Una aplicación moderna de gestión de tareas desarrollada con las mejores prácticas de Angular

Descripción

Este proyecto es un ejercicio académico diseñado para implementar y demostrar el patrón MVVM (Model-View-ViewModel) utilizando Angular como framework principal.

La aplicación permite a los usuarios gestionar sus tareas de manera eficiente, con funcionalidades completas de CRUD y una interfaz moderna y responsiva.

Tecnologías

Angular

Framework principal

TypeScript

Lenguaje principal

Tailwind CSS

Framework de estilos

Objetivo Académico

Implementar patrones de arquitectura de software modernos y mejores prácticas de desarrollo

Rendimiento

Optimizada para una experiencia de usuario fluida y responsiva en todos los dispositivos

Seguridad

Implementación de buenas prácticas de seguridad y manejo de datos del usuario
